# Парсинг задач из текста

## Цель
Извлечь структурированную информацию о задаче из естественного языка.

## Контекст
Пользователь отправил сообщение, которое классифицировано как задача. Нужно извлечь:
- Название задачи
- Описание (если есть)
- Время выполнения (scheduled_at)
- Время напоминания (reminder_at)
- Приоритет

## Инструкции
1. **Название** - краткое действие (глагол + объект)
2. **Время** - преобразуй относительные даты в абсолютные
3. **Напоминание** - обычно на 15-30 минут раньше или когда указано
4. **Приоритет** - определи по контексту: срочность, важность

## Примеры

**Вход:** "напомни мне завтра в 10:00 позвонить врачу"
```json
{
    "title": "Позвонить врачу",
    "description": null,
    "scheduled_at": "2024-11-19T10:00:00Z",
    "reminder_at": "2024-11-19T09:45:00Z",
    "priority": "medium"
}
```

**Вход:** "встреча с клиентом в среду в 15:30 по проекту X"
```json
{
    "title": "Встреча с клиентом",
    "description": "По проекту X",
    "scheduled_at": "2024-11-20T15:30:00Z",
    "reminder_at": "2024-11-20T15:15:00Z",
    "priority": "high"
}
```

## Временые выражения
- "завтра" = следующий день
- "в среду" = ближайшая среда
- "через час" = +1 час от текущего времени
- "вечером" = 18:00
- "утром" = 09:00

## Формат ответа
```json
{
    "title": "Краткое название действия",
    "description": "Дополнительные детали (или null)",
    "scheduled_at": "ISO datetime или null",
    "reminder_at": "ISO datetime или null", 
    "priority": "low|medium|high"
}
```
