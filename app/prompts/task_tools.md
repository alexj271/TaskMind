# Инструменты для работы с задачами

Эти инструменты доступны Chat Agent для управления задачами пользователя.

## create_task

Создает новую задачу для пользователя.

**Параметры:**
- title (обязательный): Название задачи
- description (опциональный): Описание задачи
- scheduled_at (опциональный): Дата и время выполнения (ISO format)
- reminder_at (опциональный): Дата и время напоминания (ISO format)
- priority (опциональный): Приоритет (low, medium, high)

**Пример:**
```json
{
  "title": "Купить продукты",
  "description": "Молоко, хлеб, яйца в супермаркете",
  "scheduled_at": "2025-12-06T18:00:00",
  "reminder_at": "2025-12-06T17:00:00"
}
```

## search_tasks

Ищет задачи пользователя по семантическому сходству.

**Параметры:**
- query (обязательный): Поисковый запрос
- limit (опциональный): Максимум результатов (по умолчанию 5)

**Пример:**
```json
{
  "query": "работа и встречи",
  "limit": 3
}
```

## update_task

Обновляет существующую задачу.

**Параметры:**
- task_id (обязательный): ID задачи
- title (опциональный): Новое название
- description (опциональный): Новое описание
- scheduled_at (опциональный): Новая дата выполнения
- reminder_at (опциональный): Новое время напоминания
- completed (опциональный): Отметить как выполненную (true/false)

**Пример:**
```json
{
  "task_id": "123e4567-e89b-12d3-a456-426614174000",
  "completed": true
}
```

## get_user_tasks

Получает список всех задач пользователя.

**Параметры:**
- limit (опциональный): Максимум результатов (по умолчанию 10)
- completed (опциональный): Показать только выполненные (true) или невыполненные (false)

**Пример:**
```json
{
  "limit": 5,
  "completed": false
}
```
